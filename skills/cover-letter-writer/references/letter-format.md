# Cover Letter .docx Formatting Specification

## Page setup
- **Paper size:** Letter (8.5" x 11")
- **Margins:** 1" all sides (1440 DXA)
- **Target length:** Under 1 page — the letter should never spill to page 2

## Layout

The letter follows standard formal business letter format, top to bottom:

### Sender block (top of page)
- Candidate name in Calibri Bold, 14pt, dark navy (#1F2937)
- Contact line: phone | email | city, state — Calibri Regular, 10pt, gray (#6B7280)
- LinkedIn URL — Calibri Regular, 10pt, gray

### Date
- One blank line below sender block
- Full date format: "February 24, 2026"
- Calibri Regular, 11pt, dark gray (#374151)

### Recipient block
- One blank line below date
- Company name
- City, State (if known)
- Calibri Regular, 11pt, dark gray

### Salutation
- One blank line below recipient
- "Dear [Name]," if contact name is in metadata.json
- "Dear Hiring Manager," as default
- Calibri Regular, 11pt, dark gray

### Body paragraphs
- One blank line below salutation
- Calibri Regular, 11pt, dark gray (#374151)
- Line spacing: 1.15
- One blank line between paragraphs (use separate Paragraph elements, not \n)
- No bullet points — this is prose, not a list
- No bold or italic emphasis in the body — let the words carry the weight

### Closing
- One blank line below final paragraph
- "Sincerely," — Calibri Regular, 11pt
- Two blank lines (for signature space in printed versions)
- Candidate full name — Calibri Bold, 11pt, dark navy

## Typography summary

| Element | Font | Size | Color | Weight |
|---------|------|------|-------|--------|
| Sender name | Calibri | 14pt | #1F2937 | Bold |
| Contact info | Calibri | 10pt | #6B7280 | Regular |
| Date | Calibri | 11pt | #374151 | Regular |
| Recipient | Calibri | 11pt | #374151 | Regular |
| Salutation | Calibri | 11pt | #374151 | Regular |
| Body | Calibri | 11pt | #374151 | Regular |
| Closing | Calibri | 11pt | #374151 | Regular |
| Signature name | Calibri | 11pt | #1F2937 | Bold |

## What to avoid

- No headers or section dividers in the body — it's a letter, not a report
- No bullet points or numbered lists — prose only
- No colored text or highlights in the body
- No images or logos
- No footer with page numbers (it's one page)
