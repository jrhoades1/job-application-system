# Achievements

Your quantified accomplishments, organized by category. The job application skills
pull from this file to tailor resumes, write cover letters, and prep for interviews.

## How to use this file

Replace the examples below with your real achievements. Be specific with numbers:
"Reduced latency by 20%" not "Improved performance." The more concrete, the better
the skills can match you to job requirements.

Items tagged `[learned: YYYY-MM-DD]` were discovered during the job search process.
The system adds these automatically when you mention accomplishments that weren't
originally listed.
